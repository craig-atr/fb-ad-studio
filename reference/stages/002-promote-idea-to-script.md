# Stage 002 — Promote Idea to Script

## Objective

Turn the approved concept into a locked **script**: the Meta ad copy plus a timed
beat sheet for an ~18 s vertical video. This is the blueprint the storyboard
expands.

Read the chosen concept from `work/<campaign>/01-ideas/ideas.md` and the brand
profile. Use `reference/templates/script.md`.

## 1. Write the ad copy

Following `reference/methodology.md` copy structure and the limits in
`reference/facebook-ad-specs.md`:

- **Headline** (~40 chars) — the hook line from the concept.
- **Subheadline** — the offer framing.
- **Primary text** — write **two**: a short punchy version (often wins) and a
  longer version. Front-load hook + offer so they survive ~125-char truncation.
- **Description** (~30 chars) — the urgency line.
- **CTA** — **Book Now**.

The headline hook must be the same line the landing page uses. Note that in the
script.

## 2. Write the beat sheet (~18 s, three acts)

Lay out the video as timed beats totaling 15–20 s. Follow the three-act shape:

```text
0:00–0:03  HOOK      arresting line + fast visual movement
0:03–0:12  PROOF     real laser in action, before/after, close-ups
0:12–0:20  OFFER+CTA offer overlay -> brand close, CTA Book Now
```

For each beat specify:

- time range
- intended **on-screen text** (big, muted-friendly, ≤ ~6 words)
- what's happening visually (one line)
- **source type**: `AI_CLIP`, `AI_IMAGE`, or `REAL_FOOTAGE`
- audio note (music energy; voice only if truly needed)

Keep something changing every 2–3 seconds. Make sure the real-footage beats
(laser, before/after) are the proof core, with AI beats as hook/emotion/brand.

## 3. Produce 3 hook variants

Per the methodology, list **3 alternate hook lines** for the same offer so the
user can A/B test. These become sibling scripts later if the user wants multiple
ads.

## 4. Write and present

Save to `work/<campaign>/02-script/script.md`. Summarize the beat sheet and copy
for the user.

## Review gate

Get explicit approval of the script (copy + beats + source tags) before stage
003. Revise in place on feedback. Do not storyboard an unapproved script.
